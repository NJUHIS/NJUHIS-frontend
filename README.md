# NJUHIS-frontend

---

## 环境

```
node >=12
```

# #安装

```
cd app && npm install
```

## 开发HMR/预览

```
 cd app && npm run serve
```

## 打包

```
cd app && npm run build
```

## 风格化

```
cd app && npm run lint
```

---

## 流程

clone下来后 先**安装**，开发时用**HMR**开启前端服务器，需要部署时**打包**，并把`dist`文件下内容部署到服务器对应文件夹下

## 注意⚠️

每次开发完成后，git提交前，务必执行**风格化**
